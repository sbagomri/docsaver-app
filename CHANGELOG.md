# Changelog

All notable changes to DocSaver will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project setup with Clean Architecture
- Riverpod state management integration
- Drift database with encryption support
- Theme system (Dark/Light mode)
- Database schema for documents, categories, reminders
- Error handling system with Result type
- Domain entities for documents, categories, reminders
- Basic home screen UI with bottom navigation
- Default categories (Passports, IDs, Driver Licenses, etc.)
- Documentation (README, SETUP, API_DOCS)

### In Progress
- Document upload flow (camera, gallery, file picker)
- OCR integration with Google ML Kit
- Category management
- Search functionality

### Planned
- Smart data extraction
- Firebase cloud sync
- Biometric authentication
- Smart reminders
- Document sharing
- Batch operations

- Clean Architecture folder structure
- Core theme and constants
- Database setup with Drift
